# Spend-IQ

Spend-IQ is a modern expense tracking and analytics dashboard that helps users
monitor, categorize, and analyze their spending efficiently through interactive
visualizations and insights.

## 🚀 Features
- Expense tracking with category-wise breakdown
- Interactive dashboards and charts
- Clean and responsive UI
- Optimized data handling for better performance

## 🛠 Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Chart.js
- SQL (for analytics queries)

## 📦 Installation & Setup

```bash
git clone https://github.com/08nitesh/Spend-IQ.git
cd Spend-IQ
npm install
npm run dev
